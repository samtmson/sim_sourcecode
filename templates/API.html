Stock Market API

An API for getting stock data on these companies(Apple, Adobe, Amazon, Activision, Comcast, Costco, Ebay, Facebook, Fox, Google, Hasbro, Intel, Microsoft, Netflix, Pepsico, Paypal, Starbucks, T-Moblie, tesla)

  • Enter /company tickersymbol to get the current share price of that copmany
  • Enter /all to get all of the share price data for every included company
  • Enter /all/company tickersymbol to get all the share price data for that company

(All data is returned in a JSON format including the date tickersymbol and share price of the company)
